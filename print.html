AI洞察日报 2026/1/3

AI 日报

今日AI资讯

  1. Codex在AI领域取得了显著进展，用户表示“在早晨醒来看到 Codex 通宵达旦的进步，感觉很棒”。
  2. 用户可以通过在 Codex 中输入“$”并进行自动补全来显式调用某个技能。开发者表示，随着 Codex 代理的增强，团队正处于一个可以进行更多创新的新阶段，并且在1月份还将推出更多创新功能。
  3. 文章建议，任何教授 Vibe Coding 的课程都应首先向学员传授计算机基础知识。
  4. “Vibe Coding”的开发成本极低，仅用一下午时间就开发出了一款名为 Vibe Diary 的产品，该产品支持AI分析、生成标签、判断待办事项和稍后阅读意图分类，并支持将所有日记导出为Markdown文档，方便与顶级AI交互，同时支持PWA应用，提供良好的手机使用体验（目前需要邀请码）。该产品借鉴了马伯庸和Karpathy的日记记录方式，可以与AI很好地结合。
  5. 当 Codex 需要花费10分钟以上处理一项任务时，通常意味着它将产生令人惊叹的成果。
  6. “Kling 2.6 Motion Control”与 Nano Banana Pro 结合，可以制作出有趣的TikTok直播风格视频。通过准备模型图片和原始视频，即可轻松生成动态视频，且只需更改原始视频即可制作不同风格的视频。
  7. “Kling 2.6 Motion Control”可以在一分钟内制作出时下流行的“可爱宝宝酷炫舞”视频。
  8. Codex 帮助用户将更多精力投入到更高层次的工作中，而无需被琐碎的细节所困扰。用户表示，自使用 Codex 以来，其工作流程发生了巨大变化，且对此非常满意。
  9. GeminiApp 询问用户新年的第一个Gemini提示是什么。
  10. 用户分享了一种低成本的AI辅助学习方法：使用便宜的支架将手机摄像头对准屏幕或书本，然后打开“豆包App”进行视频通话，可以随时提问，并陪伴一起浏览Twitter或读书。
  11. 文章将 Claude 模型 与 Claude Code 的关系比作剑客与其武器，而 GPT 使用 Codex CLI 则如同刀客使用宝刀。其他模型如GLM、Kimi、DeepSeek V3 虽然没有自己的“宝剑”，但通过使用 Claude 的“剑”也展现出了不俗的威力。
  12. Codex 能让代码库焕发生机。
  13. 另一项对 Kling Motion Control 的测试显示，该技术在处理某些场景时效果显著，尽管并非所有场景都完美，但实验过程非常有趣。
  14. 文章指出，在2026年，可以使用 notebooklm 的“fast research”或“deep research”功能自动收集资料，取代传统手动谷歌搜索的方式。对于主题研究，80%的资料可以通过“deep research”完成，其余20%则可由用户自行挑选，包括个人笔记和阅读库中的资料。同时，还提供了关于 notebooklm 的详细干货笔记和演示视频。

01-01-日刊

AI洞察日报 2026/1/1

AI 日报

今日AI资讯

  1. Google 的 Nano Banana 模型表现亮眼： 2025年对Google的图像生成和编辑模型Nano Banana来说是极其重要的一年。该模型于8月首次亮相，并迅速成为全球评分最高的图像编辑模型。11月，Nano Banana Pro发布，它基于Gemini 3 Pro，利用Gemini先进的推理和现实世界知识，实现了前所未有的信息可视化能力。Google对用户创意性地使用Nano Banana感到非常兴奋，并分享了年度最有趣的Nano Banana趋势。

  2. KlingAI推出“年度回忆”特效： KlingAI发布了基于Kling 2.5 Turbo的**“年度回忆”特效**，旨在帮助用户摆脱传统的视频制作模板，轻松制作2025年的回顾视频，并保存珍贵的回忆。

  3. Kling Motion Control引领AI视频新高度： Alex Patrascu称赞Kling的Motion Control是2025年AI视频领域的重大突破，其精准度达到了前所未有的水平。结合NB Pro，Motion Control能够实现更多可能性。

  4. Gemini推出“无提示词”挑战： Gemini本周的提示词挑战取消了提示词的要求，允许用户直接在参考图像上进行涂鸦，以达到理想效果。用户被鼓励展示如何通过这种方式结束2025年，例如创作佩戴新年眼镜的形象、展望2026年的愿景或发送年终寄语，并分享前后对比图。

  5. 罗永浩与豆包的爆笑互动： 一段视频展示了罗永浩与豆包的精彩互动，现场气氛十分热烈。

  6. Nano Banana Pro与Kling AI结合的应用案例： 一则推文展示了如何利用Nano Banana Pro和Kling AI创建一张塔可（Taco）的爆炸图。该图以商业食谱风格呈现，精确展示了五种食材（番茄莎莎酱、切达奶酪、烤鸡肉、生菜、玉米饼）的分离和垂直排列，并配有清晰的图示标注，包含食材名称和精确重量，整体风格简约、教学化且商业感十足。

  7. Kling AI的运动捕捉技术备受赞誉： madpencil_强调了Kling AI的运动捕捉技术的优势，不仅能复制视觉内容，还能智能预测并重建完整、连贯且物理真实的人体运动，即使在参考视频中身体大部分不可见的情况下，也能呈现出逼真的人类动作。

  8. Kling O1在定格动画制作中表现出色： NOBU分享了使用Kling O1制作定格动画的体验，认为其效果非常好，能够实现镜头虚化、通过提示词控制画面布局，并能自动调整以匹配角色风格，整体氛围感佳。

  9. Nano Banana Pro绘制卡通人物的经验分享： 该推文提供了一些使用Nano Banana Pro绘制**真人（卡通风格）**的经验。建议首先使用Sora将真人转换为卡通风格，并分享了关于处理未被训练过的人物、使用名字而非全名以及“多抽卡”等技巧。此外，还提供了将文章故事转化为漫画脚本的提示词参考，并附上了相关对话和绘图示例链接。

  10. Gemini协助优化README文件： 推文提及用户持续更新项目，并让Gemini协助装修了项目的README文件。同时，另一位用户“歸藏(guizang.ai)”表示终于创建了一个GitHub仓库来存放自己创建的提示词。
